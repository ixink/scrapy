# sracpy
Web Scraping Tool

How to Install:
# Windows Installation

Open Windows Powershell


Git clone

            git clone http://github.com/ixink/scrapy

Enter Directory/

           cd scrapy

Install dependencies

           pip3 install -r requirements.txt

Run app:

          python3 app.py

Open Browser and run

        http://127.0.0.1:5000

Done Ready to Go....

# Linux Installation

Open Terminal

Git clone

            sudo git clone http://github.com/ixink/scrapy

Enter Directory/

           cd scrapy

Install dependencies

           pip3 install -r requirements.txt

Run app:

          python3 app.py

Open Browser and run

        http://127.0.0.1:5000

Done Ready to Go....
